<img width="1916" height="1018" alt="image" src="https://github.com/user-attachments/assets/42d89cba-ba20-446e-bbd7-1c657a07847c" />🖥️ System Audit Dashboard

A Python Tkinter-based desktop application that combines multiple system audit tools into a single, easy-to-use dashboard. It provides real-time monitoring, dark/light mode, and can even generate audit reports automatically. An .exe version is also available for convenient use without Python installation.

🚀 Features

🌗 Dark & Light Mode — switch themes effortlessly for better visibility.

🧩 Real-time System Change Tracking — view system activity updates directly in the terminal.

📄 Audit File Generation — automatically creates a detailed audit log for future reference.

⚙️ Standalone Executable — includes an .exe build for one-click access without Python.

🖱️ Simple UI — user-friendly Tkinter interface for efficient monitoring.

🛠️ Tech Stack

Language: Python

GUI Framework: Tkinter

Modules Used:

os

platform

datetime

subprocess

tkinter (and its themed widgets)

📦 Installation
Option 1 — Run with Python

Clone this repository:

git clone https://github.com/yourusername/system_audit_dashboard.git
cd system_audit_dashboard


Install dependencies (if any):

pip install -r requirements.txt


Run the app:

python main.py

Option 2 — Run the Executable

Download the latest .exe file from the Releases section.

Double-click to launch — no Python required!

📁 Output

Terminal shows live system changes.

An audit log file (e.g., system_audit_report.txt) is generated automatically in the project folder.
